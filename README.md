Login MVC en PHP

Este proyecto es un sistema de inicio de sesión desarrollado en PHP utilizando el patrón MVC (Modelo – Vista – Controlador).

Permite autenticar usuarios contra una base de datos MySQL usando PDO y contraseñas cifradas.

Funcionalidades

Login con usuario y contraseña

Dashboard protegido (solo usuarios logueados)

Logout seguro

Contraseñas con password_hash() y password_verify()

Protección CSRF

Sesiones seguras
